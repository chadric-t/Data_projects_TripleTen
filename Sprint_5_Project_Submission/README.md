# Storytelling With Data - Returns Analysis Project

## Project Overview

This project analyzes return patterns in retail data using Tableau to identify key root causes and provide actionable insights for reducing return rates.

## Tableau Public Dashboard

**Live Dashboard Link:** \[https://public.tableau.com/app/profile/chadric.thomas/viz/TheRealSprint5Project/StoryPointPresentation?publish=yes]

## Project Components

### 1\. Data Analysis \& Visualizations

* Scatterplot showing correlation between total sales and total returns
* Bar chart displaying return rate by product category
* Geographic map showing return rate by state/region
* Time series analysis of seasonal return patterns
* Composite charts analyzing multiple factors affecting returns

### 2\. Dashboard Design Process

* **Design Requirements:** Documented dashboard specifications
* **Low-Fidelity Mockups:** Three variation sketches exploring different layouts
* **Dashboard Template:** Finalized design template with key components

### 3\. Tableau Story Presentation

* **Story Arc:** Four-point narrative structure

  * Summary of returns analysis and measurement approach
  * Dashboard component overview and interpretation guide
  * Interactive demonstration of filters and root cause identification
  * Conclusion with actionable next steps

* **Presentation Recording:** PDF File of Story
